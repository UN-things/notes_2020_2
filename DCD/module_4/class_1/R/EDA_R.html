Se tienen las variables

  • BOROUGH: Vecindario donde ocurri el accidente
  • COLLISION_ID: ID del accidente
  • CONTRIBUTING FACTOR VEHICLE (1, 2, 3, 4, 5): Razones del accidente
  • CROSS STREET NAME: La calle cruzada ms cercana en la que pas el accidente
  • DATE: Fecha del accidente
  • TIME: Hora del accidente
  • LATITUDE: Latitud del accidente
  • LONGITUDE: Longitud del accidente
  • NUMBER OF (CYCLISTS, MOTORISTS, PEDESTRIANS) INJURED: Nmero de heridos de cada tipo
  • NUMBER OF (CYCLISTS, MOTORISTS, PEDESTRIANS) KILLED: Nmero de muertos de cada tipo
  • ON STREET NAME: Calle de accidente
  • VEHICLE TYPE CODE (1, 2, 3, 4, 5): Tipo de vehculo involucrado en el accidente
  • ZIP CODE: Cdigo zip del accidente

Han incrementado el nmero de accidentes en el periodo de observacin?

Para hacer ste anlisis primero debemos cambiar el formato de la columna de la fecha para manejarla apropiadamente.

accidentes$DATE <-as.Date(accidentes$DATE,'%m/%d/%Y')

Con esta transformacin podemos extraer el mes de la fecha y hacer un anlisis por meses. Qu podemos concluir del nmero de accidentes por mes?

accidentes$m_y <- strftime(accidentes$DATE,'%y-%m')
accidentes_mes <- as.data.frame(accidentes %>% count(m_y))
ggplot(accidentes_mes, aes(x=m_y, y=n,group=1)) +
  geom_line()+xlab("Mes")+ylab("Accidentes")

Existen patrones horarios en los accidentes?

accidentes$TIME <-strptime(accidentes$TIME,'%H:%M')

Al igual que la fecha podemos transformar el formato de la hora y crear una variable que contenga nicamente la hora del accidente

accidentes$HORA <- strftime(accidentes$TIME,'%H')
accidentes_hora <- as.data.frame(accidentes %>% count(HORA))
ggplot(accidentes_hora, aes(x=HORA, y=n,group=1)) +
  geom_bar(stat="identity")+xlab("Hora")+ylab("Accidentes")

Existen patrones en los accidentes segn el da de la semana?
