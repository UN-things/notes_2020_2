Para analizar estos datos en R vamos a utilizar la librera arules y su funcin apriori para generar las reglas de asociacin. Para esto debemos transformar los datos, ya que la funcin que utilizaremos slo recibe datos de tipo transaccin. El resultado debe llevar a una fila por cada transaccin y una columna que grupe todos los tem separados por comas.

In [6]:
trans <- function(){
  lista = list()
  for (i in 1:nrow(compras)) {
    lista[i][1]=as.character(compras[i,1])
    j <- 2
    while(j <= ncol(compras) & compras[i,j] != ""){
        lista[[i]][j]=as.character(compras[i,j])
        if (j < ncol(compras))
          j = j + 1
        else
          break
    } # end while
  } # end i loop
  return(lista)
} # end function

Conclusiones¶

  • Las tablas de datos transaccionales requieren de un manejo diferente a las tablas tradicionales y requieren de transformacin en la mayora de casos para su anlisis

  • Conocer los tems ms frecuentes permite sacar conclusiones interesantes sobre las compras

  • La aplicacin de las reglas de asociacin es bastante sencilla
