Análisis de componentes principales¶

Este caso prctico muestra un ejemplo de cmo hacer un anlisis de componentes principales en R La meta del caso es conocer una de las funciones ms usadas en R para hacer anlisis de componentes principales. Entender las salidas que ofrece y tomar decisiones de anlisis basadas en ellas.

Contexto: El sector de los bienes races es uno de los ms exitosos en la economa del mundo. La gran mayora de personas en el mundo deciden rentar sus viviendas por comodidad o economa.

Problema de negocio: Existen relaciones fuertes entre las variables a considerar cuando se busca un inmueble para rentarlo? Si es as cmo podra resumir esta informacin para lograr encontrar un lugar de renta

Empezaremos por cargar las libreras requeridas para este caso

Librerías utilizadas¶

  • pandas: Es una biblioteca de software escrita como extensin de NumPy para manipulacin y anlisis de datos para el lenguaje de programacin Python, ofrece estructuras de datos y operaciones para manipular tablas numricas y series temporales.
  • seaborn: Es una biblioteca de visualizacin de datos de Python basada en matplotlib. Proporciona una interfaz de alto nivel para dibujar grficos estadsticos atractivos e informativos.
  • numpy: librera que da soporte para crear vectores y matrices grandes multidimensionales, junto con una gran coleccin de funciones matemticas de alto nivel para operar con ellas.
  • matplotlib: Es una biblioteca completa para crear visualizaciones estticas, animadas e interactivas en Python.

El anlisis de componentes principales es sensible a las dimensiones. Es recomendable siempre hacer una estandarizacin de las variables a utilizar para que las unidades de anlisis no interfieran en el anlisis. La idea es llevar todas las variables a la misma escala, as esta no interfiere con los resultados del anlisis. Esto lo podemos hacer transformando a las variables para que tengan media cero y varianza 1. Hagamos la estandarizacin del valor de la renta:

In [9]:

# Calculamos la desviacin estndar
ds_renta = data_acp['rent'].std()
# a cada valor restamos la media y dividimos por la desviacin estndar
data_acp['renta_std1'] = (data_acp['rent']-media_renta)/(ds_renta)
